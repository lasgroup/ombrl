# COMBRL: Scalable and Efficient Exploration for Continuous-Time RL
This is an open-source implementation of the COMBRL algorithm from our paper [Link with title](https://example.org). More info to come!

## Installation:

1. First install the jaxrl library from tha attached folder with pip.
2. Install maxinforl after that.
3. Install this repo.
4. Maybe, install JAX[GPU] manually.
